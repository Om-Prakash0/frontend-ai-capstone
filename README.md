# Frontend AI Capstone

This repository is created for the FlyRank Frontend AI Engineering Internship.

## Tech Stack

- React
- Vite
- JavaScript
- Node.js

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Setup

```bash
git clone https://github.com/Om-Prakash0/frontend-ai-capstone
cd frontend-ai-capstone
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.
